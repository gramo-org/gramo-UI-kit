Installation

1. With Bower

                  $ bower install gramo-ui-kit

2. Download the CSS file

We recommend using Bower to manage the UI kit, but it's also possible to download the CSS file and assets from Github.

Download CSS file and assets from Github

Grid

Simple flexbox grid based on scss-flex-grid

  • Responsive, mobile first approach
  • Define specific colum size or use a flex column (a column that grows to fill space)
  • Define offset to push column a specific number of columns.
  • Wrap columns in a '.row' and the '.row' in a '.container'

Example 1 - flex column

<div class="container grid-test"> <div class="row"> <div class="col-xs"> </div> <div class="col-xs"> </div> </div> </div>
.col-xs (flex column)
.col-xs (flex column)

Example 2 - fixed width column

<div class="container grid-test"> <div class="row"> <div class="col-sm-4 col-xs-12"> .col-sm-4 .col-xs-12 </div> <div class="col-sm-4 col-xs-12"> .col-sm-4 .col-xs-12 </div> <div class="col-sm-4 col-xs-12"> .col-sm-4 .col-xs-12 </div> </div> </div>
