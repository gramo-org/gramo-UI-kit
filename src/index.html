Installation

1. With Bower

                  $ bower install gramo-ui-kit

2. Download the CSS file

We recommend using Bower to manage the UI kit, but it's also possible to download the CSS file and assets from Github.

Grid

Simple flexbox grid based on scss-flex-grid

  • Responsive, mobile first approach
  • Define specific colum size or use a flex column (a column that grows to fill space)
  • Define offset to push column a specific number of columns.
  • Wrap columns in a '.row' and the '.row' in a '.container'

Example markup

                  
  <div class="container">
    <div class="row">
      <div class="col-xs">
        .col-xs (flex column)
        </div>
      <div class="col-xs">
        .col-xs (flex column)
      </div>
    </div>
  </div>
